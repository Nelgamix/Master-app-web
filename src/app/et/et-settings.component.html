Cours

Les cours peuvent être de 4 types:

  • Actif: le cours vient d'ADE, il est actif sur l'emploi du temps.
  • Caché: le cours vient d'ADE, il est caché (=> ne compte pas dans les statistiques); cependant les cours cachés restent visibles en opacité réduite sur l'emploi du temps (utile pour les cours qu'on veut surveiller sans les compter dans l'emploi du temps). Si l'option "cacher les cours passés" est activée, alors les cours passés (des semaines/jours précédents) seront automatiquement dans cet état.
  • Supprimé: le cours vient d'ADE, mais est complètement caché de l'emploi du temps. Il n'apparaîtra pas dans les statistiques, ni dans l'emploi du temps. Utile pour les cours qui ne vous concernent pas (pour les options par exemple).
  • Perso: le cours vient de l'utilisateur. Ces cours ne peuvent pas être cachés, ni supprimés, car ils sont définis par l'utilisateur.

Cours Perso

Les cours perso sont des cours qui ne sont pas renseignés dans l'emploi du temps, car il ne concernent qu'une seule personne, ou qu'ils sont externes à la formation.
Ces cours sont configurables depuis une pop-up ouvrable au dessus de l'emploi du temps.
Chaque cours perso possède une récurrence (UNIQUE, JOUR, SEMAINE), à laquelle le cours viendra se rajouter.
Chaque cours perso possède aussi une heure de début et une heure de fin (un cours perso ne peut pas être sur deux jours).
Le reste des éléments (nom, description, type, professeur, lieu) sont des éléments qui sont utilisés comme pour les cours réguliers.

Exclusions

Pour exclure un type de cours rapidement via les exclusions, un click droit sur le cours ouvre un menu.

Il suffit alors de sélectionner une option et le cours sera automatiquement retiré de l'emploi du temps.

Sauvegarde

Les données suivantes sont sauvegardées dans les cookies:

  • Options
  • Exclusions
  • Cours persos

Si les cookies sont inaccessibles, ou que la fonctionnalité n'est pas dans cette liste, alors la sauvegarde n'aura pas lieu.
